Subnet Calculator

A subnet calculator application that I developed using Java. The application allows users to input an IP address and subnet mask to calculate various subnetting details such as network address, broadcast address, number of hosts, and more.

The calculator is capable of handling CIDR and VLSM subnetting, making it a versatile tool for network administrators and IT professionals. The user interface is designed to be intuitive and user-friendly, ensuring that users can easily navigate and utilize the application's features.
